## README



>  本实验在小组成员的主机之间，根据选定的虚拟网络拓扑图建立虚拟连接；并且在建立的虚拟网络中选用不同的路由算法（LS 或 DV），生成网络中的路由信息，实现主机之间的数据传输。实验主要分为两部分，一部分是自组织的路由，另一部分是中心化的路由。



### Virtual Routing 文件结构

- 源代码：实验源码和可执行文件；
- 虚拟路由实验报告：整个实验的实验报告；
- 视频演示：自组织和中控路由的视频演示；
- 组员心得：组员们的心得。
